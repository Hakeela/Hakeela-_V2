# Hakeela V2

Marketing website for **Hakeela** — a social enterprise building the future of tech
inclusion by training and equipping young Africans from marginalized backgrounds and
people living with disabilities with tech skills, using AI and empathy.

The site presents Hakeela's mission, initiatives, impact, and team across a landing
page, blog, and about page.

## Tech Stack

- **React** (18) + **Vite** — frontend framework & build tooling
- **React Router** — client-side routing (Home, Blog, About)
- **Swiper** — team members carousel
- Plain CSS with design tokens (fonts: Montserrat + Bricolage Grotesque)

## Getting Started

```bash
cd frontend
npm install
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # production build
```

## Structure

```
frontend/
  public/        # brand assets & images
  src/
    components/  # Navbar, Hero, Footer, Team, CTA, etc.
    pages/       # Home, Blog, About
```
